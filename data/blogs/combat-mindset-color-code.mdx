## 1. 戰鬥心態色碼

戰鬥心態色碼，是由退役的美國海軍陸戰隊中校[Jeff Cooper](https://en.wikipedia.org/wiki/Jeff_Cooper#Combat_mindset_and_the_Cooper_color_code)發表在2006年出版[書籍](https://www.amazon.com/Principles-Personal-Defense-Jeff-Cooper/dp/1581604955?&linkCode=ll1&tag=strengthenyself-20&linkId=0da3f3ce8be67562f484085726dc885c&language=en_US&ref_=as_li_ss_tl)的的思維工具，它是用來幫助我們在採取行動，特別是**動用武力之前，做好心理準備**。

在防身情境中會執行的行動，通常具有一定攻擊性，而且**不可逆**。做出這些行為後，會創造出警戒與敵意的氛圍，並徹底改變後續與對方的互動模式。

這些行為攻擊性（或激烈程度）從輕到重，包括：全力逃跑、進行口語情勢降溫、大聲命令喝斥對方、用[手電筒](/blogs/tactical-flashlight)照向對方、朝對方噴[辣椒噴霧](/blogs/pepper-spray)。更嚴重，就是開始肢體拉扯、揮拳抱摔，甚至是抽出小刀棍棒，或拔槍開火。

開槍打死一個人，絕對不可能再挽回；拿出武器威脅或使用，就不可能再用談話解決紛爭；決定逃跑，也就沒有道理又要再回到現場。在要做出以上不可逆的行為之前，我們的大腦需要處理大量的資訊，因此，能處在合適的心理狀態下，才能幫助我們果決的做出恰當的行為，以應對衝突與戰鬥。


## 2. 牴觸感

我們在看到暴力事件的新聞後，也許會變得義憤填膺，也往往會覺得如果自己在現場，一定會作出反擊。但這其實是**後見之明**，我們是在事後已知對方是歹徒，來重新思考這個情境的，既然知道這個人就是要來傷害自己的，當然會義無反顧的進行反擊。

但問題是我們不會在事前知道對方懷有惡意，對方也不會主動告知自己是壞人。在現場終究只能靠自己的主觀，去判斷對方的惡意程度，現場的資訊不可能像事後分析時那樣客觀清晰。

因此，即使是受過訓練的人，在要傷害他人前，仍需要做好心理準備，才能**克服心中的牴觸感**。
解釋牴觸感的說法很多，例如這是來自動物本能，自然界中同一物種間的競爭通常不會往死裡打，避免物種因為內鬥內耗，反而在與不同物種間的競爭中失利。即使人類是地球上最優勢的物種，我們身上仍然保留這種本能。

牴觸感也可以說來自於道德、社會期望，大部分的文明裡，都會譴責傷害、殺死另一個人的行為，從小受到這樣的道德教育的我們，自然也會對傷害、殺死他人感到牴觸。

牴觸感也可以說是來自於大腦需要處理過量資訊，因為這些行為與決策，和我們的生存有關，必然**充滿壓力**，而且**伴隨著大量資訊**，這些資訊還包含很多雜訊、主觀成分，需要過濾，這都需要靠我們的大腦來處理。

在那樣的情境裡，我們還需要不斷保持觀察、評估威脅、並全力思考怎麼保護自己與重要他人。如果要使用具致命性的武器，又有更多需要考量的問題，像是武器的操作、會不會殺錯人？會不會誤傷他人？身處這樣的高壓情境是很不舒服的，而我們的心理本能地會想要逃避令自己不舒服的事物，所以產生了牴觸感。

總之，牴觸感是存在的，也是很正常的反應，會感到牴觸並不代表我們是懦夫、是不合格的戰士。也因此，我們更需要透過戰鬥心態色碼，讓自己的心智做好準備！

## 3. 替大腦換檔

就好像車子無法直接從靜止馬上達到最高速度，中間是需要逐漸加速、配合引擎換檔的過程，才能透過適當的齒輪比，輸出最大功率。人的警戒心、精神狀態也是同理：牴觸感，就是好像引擎的慣性；而心態色碼，就是大腦的引擎檔位。

開車時，如果一下就要從空檔直接打到最高檔位加速，絕對是不行的。這就好像在毫無防備的情況下，要應對突然出現的威脅，哪怕是受過專業戰鬥訓練的軍警人員，精神上也很可能會因為突如其來的大量資訊，產生很大的動搖，一時反應不過來，沒有辦法馬上做出最好的應對。

更別提一般人，很可能就是腦袋直接當機，愣在原地不知所措，或者陷入歇斯底里的恐慌(panic)狀態，根本無法有效應對威脅，也無法保護自己和重要他人。

既然我們要認真思考防身，那就不能變成監視器畫面裡常見的那種愣在原地的狀態！以下就來詳細介紹心態色碼中各個顏色的意義！

## 4. 色碼介紹

在最原始Cooper提出的版本裡有四個色碼，**戰鬥準備**的程度由低到高，依序是**白、黃、橘、紅**。美國海軍陸戰隊後來在色碼中又加上**黑色**，所以有些資料來源裡，會有五個色碼。

通常來說，戰鬥準備和**警戒心**是有正相關的。我們正是因為察覺到危險，才要做出戰鬥準備。警戒的等級越高，通常代表準備要做出的行為，攻擊性越強烈。

在解釋色碼時，我會兼顧「警**戒**」和「預**備**」，說明這些色碼代表的**戒備程度**。

<h3 id="白色" style={{ color: '#9CA3AF', scrollMarginTop: "4em" }}>白色</h3>

**毫無戒備**的狀態。這個狀態下，意識無法處理瞬間湧出的大量資訊，牴觸感最強烈，難以採取會傷害他人的行為。

在這個狀態下，通常也代表我們對周遭變化一無所知，毫無警戒。這可能是我們正在睡覺，或是走在路上專注在滑手機的時候。我們要盡量避免處於這個狀態，就算在家裡也一樣！

<h3 id="黃色" style={{ color: '#FACC15', scrollMarginTop: "4em" }}>黃色</h3>

**放鬆戒備**的狀態。心態上，我們**接受世界上存在危險與惡意**的事實，也**做好危險如果發生，我們得做些什麼**的心理準備。所以當衝突真的發生時，我們的意識能夠快速的接受現況並行動，而不是原地愣住。

從警戒角度來看，這個狀態並沒有針對特定目標，但我們主動保持對周遭的戒備與觀察，持續的接收、分析周遭資訊。有意認真防身的人，應該讓自己**隨時保持在這個狀態**。

這與草木皆兵的狀態不一樣，並不是要我們去敵視周圍的所有人。我們要做的，就只是接受「惡意是存在的」這個事實，並在這個前提下，照常過我們的生活。

<h3 id="橘色" style={{ color: '#F97316', scrollMarginTop: "4em" }}>橘色</h3>

**有對象的戒備**。心態上我們有指定對象，且認定對方試圖傷害自己，所以做好要採取激烈行為的心理準備，但距離實行，就只差一步。我們也會將注意力集中在指定對象身上，仔細警戒著他的一舉一動。

我們通常是還在黃色狀態，廣泛的觀察周遭時，就已經留意到對象的存在。而他多半是做出了一些異常的行為，才會讓我們決定進入到橘色狀態。

這個異常行為不需要任何客觀標準，只要自己覺得有異，就可以主觀認定對方有惡意。對方可能穿著破爛邋遢、可能是特定人種、可能是穿著或說話讓我們有不舒服的感覺，不用等到對方主動顯露出敵意，就可以先認定對方試圖傷害自己，然後做好心理準備。


這時候的做好準備，不只是在精神上下定決心，我們可以用**實際肢體行為**做好準備。例如把手伸到口袋裡，握住隨身攜帶(EDC)的辣椒噴霧，並找到噴嘴方向，拇指放在按鈕上，確保一從口袋拿出來馬上就能向前噴。

利用肢體上的實際作為，反過來影響精神層面，讓自己真正調整好備戰的心態。如果想要逃跑，也可以開始規劃路線，以便自己等下要快速衝刺離開現場。

我們不會因為在心裡認定對方試圖傷害自己，需要承擔什麼代價。就算真的是誤會了，只要我們沒有付諸行動，對方不會受到實質傷害，他們也沒辦法拿我們怎樣。我們沒有理虧，不需要去擔心「別人」會覺得自己很沒有同理心、政治不正確、或是氣量狹小。

如果你真的受了委屈，我相信那個「別人」是不會一起分攤你的痛苦的，所以不用去管「別人」怎麼想！

<h3 id="紅色" style={{ color: '#DC2626', scrollMarginTop: "4em" }}>紅色</h3>

**行動狀態**。解除了心裡的牴觸感，確實執行自己決策的行為，並專注在處理當下的狀況。

對方也許顯露出了敵意，也許沒有。但總之，你決定採取行動。那色碼系統就完成了它的功能，之後不用再管什麼色碼了，專注處理現況就好。

看是要怎麼發揮機智，讓情勢降溫，化解衝突；還是打算擺低姿態道歉了事，或是要將過去花時間練習的槍法、格鬥術、武術發揮出來，這都沒有對錯，一切自己決定。

要逃跑，就全力衝刺離開現場。要戰鬥，就確實的造成對方的疼痛和傷害，有需要的話，就置對方於死地，一切以保全自己與重要他人為優先！

## 5. 不要本末倒置

上個段落提到，戰鬥準備和警戒心密切關連，也因此，有些人會拿色碼來描述**戒心等級**，或是對周遭狀態的**意識程度**(level of situational awareness)。而戒心又與觀察到的威脅相關，所以又有人會拿色碼來描述**威脅等級**。這樣無限上綱、不斷擴充原始定義的結果，就是造成混淆，甚至倒因為果。

Cooper在推廣色碼觀念的過程中，發現有些學員總會想要把觀察到的威脅指標強行分類到某個色碼中，但這只是自己把問題複雜化，然後被自己給搞糊塗。

別忘了，色碼是為了幫助自己做好戰鬥準備，而不是用來衡量危險。因果關係上，是自己根據觀察到的威脅，**先決定好要採取的行為**，然後**才用色碼**進入對應的心理狀態，幫助自己**對抗牴觸感**，進而順利的執行決策。

而不是反過來，變成先依據危險程度，才決定自己要處在什麼心理狀態裡，最後才採取對應的行為。這麼做只會讓自己陷入被動，只能按照對方展現出來的行為，決定自己的回應，這樣就本末倒置了！


## 6. 練習

現在我們知道戰鬥心態色碼，以及它的使用目的是什麼。下一個問題，是要怎麼**有效的練習、實踐**？

要保持在黃色心態，相信對大多數人是比較容易的。這個狀態裡需要練習的反倒是怎麼建立有效的[Situational Awareness](/blogs/situational-awareness)（狀態意識、態勢感知），讓意識保持接收環境中的資訊。

要實踐戰鬥心態色碼，我認為關鍵在於讓自己能夠**熟練地做出紅色狀態中的行為**，先能夠實踐這些激烈的行為，才會知道能怎麼做好準備（橘色狀態）。而想把一件事情變熟練，最有效的辦法，就是**大量重複**這些行為。

如果你選擇以**辣椒噴霧**作為主要防身手段，那就去找一個固定目標試用，去親身感受噴霧的射程、在空中的飛行軌跡，要怎麼快速解除保險。也別忘了去實測自己能多快的從口袋或包裡抽出噴霧，並拿練習用噴霧（內容物不含辣椒素），進行實際的對人練習。

如果決定碰到事情要**逃跑**，那就確保自己有足夠的體能和腳程，可以全力奔跑至少3分鐘。

如果去練**武術、格鬥**，就要充分與人練習對打、對練，讓自己習慣往對方的身體揮拳踢腿、把對方摔砸到地上、絞勒對方關節，也習慣閃躲、承受、化解對方的這些攻擊。

如果你人在美國，也**買了槍**，就要到靶場充分練習。當然，槍不能直接對人練習，但我們可以充分練習槍械的操作，學習正確的瞄準、穩定的扳機控制，讓自己習慣開槍的感覺。別忘了也要練習怎麼從槍套中，或是家裡放槍的抽屜裡快速取用槍械，模擬我們真正碰到狀況時會有的操作。

## 7. 結語

最後很快地做個總結。

我們之所用使用心態色碼，是要消除行動中的牴觸感，避免一時的猶豫，變成我們在戰鬥中的劣勢。能夠順利的從日常狀態過渡到戰鬥狀態，我們也會在狀況中更能保持冷靜，並做出更好的反應與決策，提升生存機率！

除了防身情境，這樣的戰鬥思維工具，也可以應用到職場、商場、人際關係裡，幫助我們在行動前切換好心理狀態。就算你所處環境裡沒有那麼多「暴力」，但我敢說「衝突」仍無所不在。學習防身思維，不僅是應對未來的不時之需，在日常的應對進退裡也會派上用場，而且最重要的是，讓你**變成一個更強大的人**！
